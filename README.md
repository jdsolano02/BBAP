## üìñ Project Overview

**BBAP (Bloom Battle Amateur Practice)** is a web application designed to assist baristas and coffee enthusiasts in training for the **2024 Bloom Battle Amateur** competition (hosted at Casa Pinto, Costa Rica).

This tool addresses the need to digitize the practice session workflow, allowing competitors to create brewing recipes, validate technical parameters, and ensure compliance with official competition rules.

### üöÄ Key Features

* **Rule Validation:** Automatic verification of parameters against the official Bloom Battle 2024 standards.
* **Recipe Management:** User interface for logging critical extraction variables (Dose, Ratio, Time, Temperature).
* **Ratio Calculator:** Automated math logic to ensure brewing consistency.
* **Responsive Design:** Fully accessible on mobile devices for direct use at the coffee bar.
